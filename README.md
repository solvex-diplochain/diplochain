## 🇧🇫 À propos du projet

DiploChain est une solution blockchain qui permet de :
- ✅ Émettre des diplômes numériques infalsifiables
- ✅ Vérifier un diplôme en 3 secondes gratuitement
- ✅ Éliminer les faux diplômes au Burkina Faso

## 🚀 Technologies utilisées

- **Frontend** : React.js / Vue.js
- **Mobile** : Flutter
- **Backend** : Node.js / Express
- **Blockchain** : Solidity / Ethereum
- **Base de données** : MongoDB
